# Student-Management-System-
A simple console-based Student Management System developed in C++ using File Handling.
This project allows users to manage student records with features like adding, displaying, searching, updating, and deleting student information.

Features:
1.Add new student records
2.Display all student records
3.Search student by ID
4.Update student details
5.Delete student records
6.Persistent data storage using file handling
7.Simple menu-driven interface
